# Assets Directory

This directory should contain the following image files:

1. **icon.png** (1024x1024)
   - App icon shown on device home screen
   - Should be a square PNG with transparent background

2. **splash.png** (1242x2436 or similar)
   - Loading screen shown when app starts
   - Should match your brand colors

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon
   - Should be a square PNG

4. **favicon.png** (48x48)
   - Web favicon for web version
   - Small square icon

## Quick Start - Placeholder Assets

For development, you can use placeholder images:

1. Create simple colored squares using any image editor
2. Or use online tools like:
   - https://placeholder.com/
   - Canva
   - Figma

3. Name them according to the list above
4. Place them in this `assets` folder

## Production Assets

For production apps:
- Use high-quality images
- Follow platform guidelines (iOS/Android)
- Test on multiple devices
- Consider different screen sizes
